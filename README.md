## 🎨 Design Philosophy

Inspired by **BellaVita** aesthetics — clean, minimal, and luxurious:

- **Rounded corners** on every element (12px–36px)
- **Glassmorphism** navbar & overlays
- **Soft shadows** for depth
- **Mediterranean color palette** blending warm Spanish terracotta, French lavender, Italian olive green, and Algerian gold
- **SVG illustrations** instead of heavy images for lightning-fast load times

---

## 🛠️ Tech Stack

```
HTML5          ─── Semantic structure & accessibility
CSS3           ─── Custom Properties, Flexbox, Grid, Glassmorphism
JavaScript     ─── Vanilla JS, IntersectionObserver, localStorage
GSAP           ─── Scroll animations & advanced transitions
Google Fonts   ─── Cairo, Playfair Display, Inter
```
